# Portfolyo

Kişisel portfolyo web sitesi. React ve Vite ile geliştirilmiştir.

## Özellikler

- **Ana Sayfa** – Hoş geldin bölümü
- **Hakkımda** – Kısa tanıtım
- **Projeler** – Yayınladığım uygulama ve projeler, Google Play Store yönlendirme butonları
- **İletişim** – İletişim bilgileri ve form

## Projeler

- **IBAN Cüzdanım** – Flutter ile geliştirilen IBAN yönetim uygulaması ([Google Play](https://play.google.com/store/apps/details?id=com.ibrahim.iban_wallet))
- **SmartStock** – Küçük işletmeler için stok ve envanter yönetim uygulaması (Flutter, Firebase, SQLite) ([Google Play](https://play.google.com/store/apps/details?id=com.devisb.stock_management))
- **Analysis and Classification System of Lateral Transfer Documents** – PDF OCR ve NLP tabanlı belge sınıflandırma sistemi

## Teknolojiler

- React 19
- Vite 7
- React Icons
- Framer Motion

## Kurulum ve Çalıştırma

```bash
npm install
npm run dev
```

Tarayıcıda `http://localhost:5173` adresinden erişebilirsiniz.

## Build

```bash
npm run build
npm run preview
```
